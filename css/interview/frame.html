Sass 是一款强化 CSS 的辅助工具，它在 CSS 语法的基础上增加了变量 (variables)、嵌套 (nested rules)、混合 (mixins)、导入 (inline imports) 等高级功能，这些拓展令 CSS 更加强大与优雅。使用 Sass 以及 Sass 的样式库（如 Compassopen in new window）有助于更好地组织管理样式文件，以及更高效地开发项目

特色功能：

  • 完全兼容 CSS3

  • 在 CSS 基础上增加变量、嵌套 (nesting)、混合 (mixins) 等功能

  • 通过函数open in new window进行颜色值与属性值的运算

  • 提供控制指令 (control directives)open in new window等高级功能

  • 自定义输出格式
