# FileReader

# 概念

FileReader 对象允许Web应用程序异步读取存储在用户计算机上的文件（或原始数据缓冲区）的内容，使用 Fileopen in new window 或 Blobopen in new window 对象指定要读取的文件或数据。

其中File对象可以是来自用户在一个open in new window元素上选择文件后返回的FileListopen in new window对象,也可以来自拖放操作生成的 DataTransferopen in new window对象,还可以是来自在一个HTMLCanvasElementopen in new window上执行mozGetAsFile()方法后返回结果。

重要提示： FileReader仅用于以安全的方式从用户（远程）系统读取文件内容 它不能用于从文件系统中按路径名简单地读取文件。 要在JavaScript中按路径名读取文件，应使用标准Ajax解决方案进行服务器端文件读取，如果读取跨域，则使用CORS权限。

Note: 此特性在 Web Workeropen in new window 中可用

# 构造函数

FileReader()open in new window

返回一个新构造的FileReader。

# 属性

FileReader.erroropen in new window 只读

一个DOMExceptionopen in new window，表示在读取文件时发生的错误 。

FileReader.readyStateopen in new window 只读

表示FileReader状态的数字。取值如下：

常量名值描述
EMPTY0还没有加载任何数据.
LOADING1数据正在被加载.
DONE2已完成全部的读取请求.

FileReader.resultopen in new window 只读

文件的内容。该属性仅在读取操作完成后才有效，数据的格式取决于使用哪个方法来启动读取操作。

# 事件处理

FileReader.onabortopen in new window

处理abort (en-US)open in new window事件。该事件在读取操作被中断时触发。

FileReader.onerror (en-US)open in new window

处理error (en-US)open in new window事件。该事件在读取操作发生错误时触发。

FileReader.onloadopen in new window

处理load (en-US)open in new window事件。该事件在读取操作完成时触发。

FileReader.onloadstart(暂无)

处理loadstart (en-US)open in new window事件。该事件在读取操作开始时触发。

FileReader.onloadend(暂无)

处理loadend (en-US)open in new window事件。该事件在读取操作结束时（要么成功，要么失败）触发。

FileReader.onprogress(暂无)

处理progress (en-US)open in new window事件。该事件在读取Blob时触发。

因为 FileReader 继承自EventTargetopen in new window，所以所有这些事件也可以通过addEventListeneropen in new window方法使用。

# 方法

FileReader.abort()open in new window

中止读取操作。在返回时，readyState属性为DONE。

FileReader.readAsArrayBuffer()open in new window

开始读取指定的 Blobopen in new window中的内容, 一旦完成, result 属性中保存的将是被读取文件的 ArrayBufferopen in new window 数据对象.

FileReader.readAsBinaryString() open in new window(不推荐)

开始读取指定的Blobopen in new window中的内容。一旦完成，result属性中将包含所读取文件的原始二进制数据。

FileReader.readAsDataURL()open in new window

开始读取指定的Blobopen in new window中的内容。一旦完成，result属性中将包含一个data: URL格式的Base64字符串以表示所读取文件的内容。

FileReader.readAsText()open in new window

开始读取指定的Blobopen in new window中的内容。一旦完成，result属性中将包含一个字符串以表示所读取的文件内容。
